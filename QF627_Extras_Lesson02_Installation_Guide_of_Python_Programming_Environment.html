Downloading Anaconda Navigator

Navigator supports the same operating systems as Anaconda Distribution. These include:

  • Windows 10 x86_64 or later

  • macOS 10.14+, 64-bit

  • Ubuntu 18+/Centos7+, 64-bit

The most current installation of Navigator also supports Python versions 3.11+.

WINDOWS [back to top]

Disclaimer: Nope you are not seeing it wrong, the subpoints are numbered deliberately. Python is a zero-indexed language. And as Pythonians from now onwards, we will speak like Pythonians.

Step 1: Download Anaconda Installer

  1. Open your web browser and go to the Anaconda Distribution page
  2. Click on the Download button.
  3. Select the Windows version.

Step 2: Run the Installer

  1. Locate the downloaded installer (usually in the Downloads folder).
  2. Double-click the installer to start the installation process.

Step 3: Install Anaconda

  1. Click Next on the Welcome screen.
  2. Read and accept the license agreement, then click I Agree.
  3. Choose whether to install for Just Me or All Users, then click Next.
  4. Select the installation location (the default location is usually fine), then click Next.
  5. Click Install to begin the installation.

Step 4: Complete Installation

  1. Once the installation is complete, click Next.
  2. Optionally, you can install Microsoft VSCode. If you want, check the box and click Next; otherwise, click Skip.
  3. Click Finish to complete the installation process.

Step 5: Launch Anaconda Navigator

  1. Open the Start Menu and search for Anaconda Navigator.
  2. Click on Anaconda Navigator to launch the application.

MAC [back to top]

Disclaimer: Nope you are not seeing it wrong, the subpoints are number deliberately. Python is a zero-indexed language. And as Pythonians from now onwards, we will speak like Pythonians.

Step 1: Download Anaconda Installer

  1. Open your web browser and go to the Anaconda Distribution page
  2. Click on the Download button.
  3. Select the macOS version. (silicon or intel)

Step 2: Run the Installer

  1. Locate the downloaded installer (usually in the Downloads folder).
  2. Double-click the .pkg file to start the installation process.

Step 3: Install Anaconda

  1. Click Continue on the Welcome screen.
  2. Read and accept the license agreement, then click Continue and Agree.
  3. Click Continue to confirm the installation location (the default location is usually fine). Check documentation if you want specific locations
  4. Click Install to begin the installation.

Step 4: Complete Installation

  1. Once the installation is complete, click Continue.
  2. Click Close to finish the installation process.

Step 5: Launch Anaconda Navigator

  1. Open Finder and navigate to the Applications folder.
  2. Find and double-click on Anaconda Navigator to launch the application.
